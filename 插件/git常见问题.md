
# 如何高效的使用 Git

## Push failed: Unable to access 'https://github.com/': OpenSSL SSL_read: Connection was reset, errno 10054
输入下面两条指令取消代理信息的配置
```
git config --global --unset http.proxy   #取消http代理
git config --global --unset https.proxy  #取消https代理
```
